This Is A Python App That Is Used As An MP3 Player And Portable PiHole Router. Its Made To Run On A Raspberry Pi 2 Model B Running DietPi With An 800x480 Screen.

Features:
- Local MP3 Playback Support
- Playlists, Album Covers, Repeat
- SOON, USB Stick Storage, Shuffle, PiHole Router, Webhost
